polymer_elements.polymer_selector library

polymer-selector is used to manage a list of elements that can be selected. The attribute 'selected' indicates which item element is being selected. The attribute "multi" indicates if multiple items can be selected at once. Tapping on the item element fires 'polymeractivate' event. Use the 'polymerselect' event to listen for selection changes. The CustomEvent.detail for 'polymerselect' is a map containing 'item' and 'isSelected'.

Example:

<polymer-selector selected="0">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</polymer-selector>

polymer-selector is not styled. So one needs to use "polymer-selected" CSS class to style the selected element.

<style>
  .item.polymer-selected {
    background: #eee;
  }
</style>
...
<polymer-selector>
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</polymer-selector>

The polymer-selector element fires a 'polymerselect' event when an item's selection state is changed. The CustomEvent.detail for the event is a map containing 'item' and 'isSelected'.
